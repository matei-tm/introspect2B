# Observability Screenshots and Metrics

This directory contains snapshots of CloudWatch dashboards, metrics, and monitoring visualizations.

## Contents

- `dashboards/` - CloudWatch Dashboard screenshots
- `metrics/` - Individual metric visualizations
- `logs/` - Log Insights query results
- `alerts/` - Alarm configuration screenshots

## Key Metrics Captured

- API latency (p50, p95, p99)
- Error rates by endpoint
- DynamoDB consumed capacity
- Bedrock token usage
- S3 request latencies
- Pod restart frequencies
- IAM IRSA access patterns
